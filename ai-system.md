# 🧠 AI Intelligence System

FaultPulse uses a hybrid AI system:

---

## ⚡ Layer 1 — Local Intelligence

* Fast
* Rule-based
* Always available

---

## 🧠 Layer 2 — AI (Groq API)

* Deep reasoning
* Dynamic explanations
* Code fixes

---

## 🔁 Reliability System

* Retry mechanism
* Caching (same error → no repeat calls)
* Fallback if AI fails

---

## 🔍 Example Output

```
Error: TypeError

Explanation:
You are trying to access a property on null.

Fix:
if (obj) obj.name;
```

---

## 💡 Why Hybrid?

AI alone is unreliable.

So system uses:

```
Local → Fast
AI → Smart
Combined → Reliable
```
